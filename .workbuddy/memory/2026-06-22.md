### 功能特性
- 左侧栏：101 篇对话列表，支持搜索过滤，显示已读进度
- 每段对话：标题、全篇播放/停止、段落数
- 每段落：播放按钮、五档语速（0.5x/0.75x/1x/1.25x/1.5x）、原文显隐切换、译文折叠
- 底部词汇表：单词发音、释义
- 键盘导航：← → 翻篇，Space 停止播放
- 本地存储：进度和语速偏好持久化
- 响应式设计：适配手机/平板/桌面

### 说明
- 该书为纯意大利语教材，不含中文译文。译文区展示英文词汇释义
- 音频使用浏览器 Web Speech API 意大利语 TTS，无需预生成音频文件
