## italian-101-conversations 网站修复

### 问题
用户反馈：音色选择功能没有生效，听不到不同男女声音

### 排查
1. 代码逻辑正确 - `getAudioByVoice()` 使用 `currentVoice` 获取对应音频
2. JSON 数据正确 - `conv_1.json` 包含三种音频数组
3. **问题根因**: 切换音色后下拉菜单没有同步刷新显示

### 修复
```javascript
function setVoice(convIdx, voiceName) {
  currentVoice = voiceName;
  localStorage.setItem('italianVoice', voiceName);
  setVoiceUI(); // 新增：同步所有下拉菜单
}

function setVoiceUI() {
  document.querySelectorAll('.voice-select').forEach(sel => {
    sel.value = currentVoice;
  });
}
```
